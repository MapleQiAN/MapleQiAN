### 🤺 About Me

<img align="right" width="88" src="https://cdn.jsdelivr.net/gh/MapleQiAN/MapleQiAN/assets/images/jobs.png" />

你好，我是 Serendy。
一名普通的北京邮电大学计算机科学与技术专业研究生。
日常生活由几块拼图组成：写代码、做研究、运动、打游戏，然后在第二天的中午笑自己。

我喜欢去探索那些看上去很复杂的系统——网络协议、操作系统、硬件指令集。
但我也同样喜欢一些小而有趣的创意，比如把日记做成可以和自己聊天的程序，
或者把丢失物品的校园平台做得温柔又实用。

对我来说，技术不是冷冰冰的命令行，它更像是另一种叙事方式。
